# K-Means Clustering and Choosing the Right k

This repository contains a tutorial on K-Means clustering and how to choose the number of clusters (`k`) using the Elbow Method and Silhouette Score. The tutorial is provided as a Jupyter Notebook, and demonstrates the method on the Iris dataset.

## Contents

- `kmeans_clustering_tutorial_arslan_jamil.ipynb`: Jupyter notebook with code and plots.
- `kmeans_clustering_tutorial_arslan_jamil.pdf`: Tutorial and explanation on `Choosing the Right k`.

## How to Use

1. Clone or download the repository.
2. Open the notebook in [Jupyter](https://jupyter.org/) or [Google Colab](https://colab.research.google.com/).
3. Run the cells step by step.

To run locally, ensure you have the following Python packages:
- numpy
- matplotlib
- scikit-learn

Install using pip if needed:
```sh
pip install numpy matplotlib scikit-learn
```
